Términos y Condiciones
Políticas de cambios y devoluciones
Proceso de compra
Conoce nuestros acuerdos
  1. Información del Producto:
    • Todos los productos publicados son fabricados por nosotros, con materiales seleccionados y calidad garantizada.
    • Las imágenes son referenciales y pueden tener ligeras variaciones en tonos debido a la iluminación o pantalla del dispositivo.
  2. Proceso de Compra:
    • Al elegir talla, color y modelo, tu pedido se enviará al WhatsApp para coordinar y registrar la orden.
    • El pedido será agendado únicamente después de confirmar el pago completo.
    • La fabricación toma aproximadamente 7 días hábiles, ya que se realiza bajo pedido o tras completar la media (cantidad mínima para producción).
    • Luego del plazo, se realiza el despacho o se coordina el recojo.
  3. Formas de Pago:
    • Aceptamos pagos por Yape, Plin y transferencias bancarias.
    • Es responsabilidad del cliente enviar el comprobante y su descripción (nombre, modelo, talla y color) correctamente.
  4. Cambios y Devoluciones:
    • No se aceptan cambios ni devoluciones, salvo fallas de fábrica visibles y considerables.
    • El reclamo debe hacerse dentro de los 2 días posteriores a recibir el producto.
    • Se debe enviar fotos y/o videos para evaluación.
    • Si la falla es comprobada, se ofrece descuento en una próxima compra o cambio, según coordinación.
    • En caso el cliente no acepte, se podrá evaluar devolución o reemplazo del producto según disponibilidad.
  5. Envíos y Recojos:
    • Realizamos envíos a todo el Perú por Shalom.
    • También puedes recoger en nuestra tienda física en Trujillo de Lunes a Sábado desde las 8 am hasta 8 pm.
    • Previo acuerdo, tu pedido puede ser juntado con futuras compras (por TikTok Live o redes) para facilitar la entrega, siempre y cuando los tiempos lo permitan.
  6. Confidencialidad:
    • Toda la información brindada por el cliente es confidencial y solo será usada para gestionar sus pedidos.
  1. No realizamos cambios ni devoluciones, excepto en casos de fallas visibles de fábrica. No se aceptan reclamos por detalles mínimos o propios del proceso de fabricación artesanal.
  2. Plazo para reclamos: El cliente cuenta con un máximo de 5 días calendario desde la fecha de pago para presentar su reclamo. Pasado este plazo, no se aceptarán solicitudes de cambio o devolución.
  3. Revisión del caso: Para iniciar el proceso, el cliente deberá enviar al WhatsApp:
    • Video y fotos claras del producto desde distintos ángulos.
    • Detalle del problema observado.
    • Comprobante de pago.
  4. Evaluación del reclamo: El equipo de RousseModa revisará cuidadosamente la evidencia. Si se determina que existe una falla real de fabricación, se ofrecerán dos alternativas:
    • a. Descuento en una próxima compra, como forma de compensación.
    • b. Cambio o devolución del producto, solo si el daño es significativo y el cliente así lo solicita.
  5. Condiciones para el cambio/devolución:
    • El producto debe estar sin uso y en perfectas condiciones, tal como fue entregado.
    • El cliente deberá devolver el producto usando la agencia Shalom o según coordinación previa.
    • Los costos de envío por devolución corren por cuenta del cliente, excepto si el error es atribuible a RousseModa.
  6. Casos no válidos para reclamo:
    • Desperfectos causados por mal uso, golpes o desgaste natural.
    • Cambios por talla, modelo o color una vez recibido el producto.
    • Productos con descuentos especiales o adquiridos en dinámicas sorpresa.
  1. Confirmación de pedido: Una vez el cliente seleccione el producto, color y talla en la página web, será redirigido al WhatsApp de atención para registrar su orden de forma personalizada.
  2. Validación y pago: El pedido será agendado solo después de haber confirmado el pago total. Sin este paso, la orden no se considerará registrada oficialmente.
  3. Tiempo de espera: Al trabajar bajo un sistema de producción por medias, el pedido ingresará a fase de fabricación, y el tiempo estimado de entrega será de aproximadamente 7 días hábiles.
    Importante: Este plazo puede variar dependiendo del avance de producción y disponibilidad del modelo.
  4. Fabricación por media: La producción inicia cuando se completa la media docena del modelo solicitado. El cliente será notificado si hubiera algún retraso excepcional.
  5. Coordinación de entrega: Una vez finalizado el proceso de fabricación:
    • El equipo de RousseModa se comunicará vía WhatsApp para coordinar el envío (a través de Shalom) o el recojo en tienda (solo Trujillo).
    • El cliente debe brindar datos completos para el despacho: nombre, DNI y sede Shalom.
  6. Cambios y cancelaciones: Una vez iniciado el proceso de fabricación, no se aceptan cancelaciones ni cambios de modelo, color o talla, ya que cada par es fabricado bajo pedido.
  7. Acumulación de compras: En caso el cliente tenga compras anteriores realizadas por otros canales (live, tienda física u otros), su envío podrá ser unificado previa coordinación directa vía WhatsApp. Esta opción está sujeta a disponibilidad de tiempos y estado de preparación de cada pedido.
